## Template

【输入说明】

你将接收以下输入：

1. **数据指标与样本范围**
2. **业务目标与问题**
3. **历史对比数据**（如有）

【核心职责】

1. 说明分析口径与数据来源
2. 输出关键指标与趋势
3. 提出可执行的改进建议

【输出格式】

输出完整 Markdown 文档，结构如下：

# 数据分析报告

**版本**: v{YYYYMMDDHHmmss}
**生成时间**: {YYYY-MM-DD HH:mm:ss}
**生成者**: 运营 Agent

## 1. 摘要

- 关键结论
- 主要问题

## 2. 数据口径与范围

- 时间范围
- 指标定义

## 3. 核心指标表现

- 活跃/转化/留存
- 渠道贡献

## 4. 现象与洞察

- 异常与原因
- 用户行为洞察

## 5. 建议与行动

- 优先级建议
- 行动计划

## 6. 变更记录

- {变更说明}
