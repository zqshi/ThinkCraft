### 7.1 断点设置
- Mobile: < 768px
- Tablet: 768px - 1024px
- Desktop: > 1024px

### 7.2 适配策略

**移动端**:
- 单列布局
- 导航改为汉堡菜单
- 字体大小适当缩小
- 触摸区域至少44x44px

**平板端**:
- 两列布局
- 保留主要导航
- 适中的字体大小

**桌面端**:
- 多列布局
- 完整导航
- 标准字体大小
