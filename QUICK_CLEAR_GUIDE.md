# 快速清理指南

## 🚀 最快方法（推荐）

### 方法1：使用可视化工具

1. **双击打开** `clear-data.html`
2. **点击** "开始清理" 按钮
3. **等待** 清理完成（约5秒）
4. **刷新** 页面验证

✅ **完成！** 所有数据已清理。

---

### 方法2：使用控制台脚本

1. **打开应用** 并按 `F12`
2. **切换到** Console 标签
3. **复制粘贴** `clear-data-console.js` 的内容
4. **按** Enter 执行
5. **点击** "确定" 确认
6. **刷新** 页面验证

✅ **完成！** 所有数据已清理。

---

## ⚠️ 重要提示

- **不可逆**：清理后无法恢复
- **备份**：清理前请备份重要数据
- **关闭标签页**：清理时请关闭其他 ThinkCraft 标签页

## 🔍 验证清理

清理后，检查：

1. **刷新页面**（Ctrl+Shift+R）
2. **对话列表为空**
3. **项目列表为空**
4. **IndexedDB 为空**（F12 → Application → IndexedDB）

---

## 🆘 遇到问题？

### 问题1：提示 "Database blocked"

**解决**：
1. 关闭所有 ThinkCraft 标签页
2. 只保留一个标签页
3. 重新清理

### 问题2：清理后还有数据

**解决**：
1. 硬刷新页面（Ctrl+Shift+R）
2. 清除浏览器缓存（Ctrl+Shift+Delete）
3. 重启浏览器

### 问题3：清理失败

**解决**：
1. 使用方法2（控制台脚本）
2. 手动清理（见完整指南）
3. 查看控制台错误信息
