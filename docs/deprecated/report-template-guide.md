# ThinkCraft 新报告格式 - 创意思维结构化分析报告

## 说明
这是基于苏格拉底式对话的六章节报告框架。

## 数据映射

由于当前是基于对话历史动态生成，需要从 `state.messages` 中提取信息：

- 原始表述：第一条用户消息
- 核心定义：AI总结的精炼定义
- 价值主张：从对话中提取
- 假设清单：从对话中识别
- 待验证问题：从对话中总结
- 行动建议：基于对话生成

注意：由于新的对话模式是自由的，不再有固定的 userData 结构，
需要通过AI分析整个对话历史来生成这六个章节。
